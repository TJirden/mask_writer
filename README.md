## **Mask-writer**  

Простое приложение для обработки файлов с помощью XOR-маскирования. Позволяет применять заданную маску к файлам, сохранять результат в указанную папку и логировать процесс.  

### **Основные функции:**  
- Выбор файла для обработки  
- Настройка XOR-маски  
- Опции:  
  - Перезапись или создание копии выходного файла  
  - Удаление исходного файла после обработки  
  - Сохранение настроек между запусками  
- Прогресс-бар и лог операций  

### **Сборка:**  
Требуется **Qt 5/6** и **CMake**.  
```sh
mkdir build && cd build  
cmake .. && cmake --build .  
```

### **Использование:**  
1. Укажите входной файл  
2. Задайте параметры обработки  
3. Запустите процесс
